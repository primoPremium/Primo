## Delegation Path
1. Strategic Direction (Primo)
   - High-level objectives
   - Success criteria
   - Strategic constraints

2. Operational Coordination (Task-Rabbit)
   - Task breakdown
   - Agent assignments
   - Timeline management
   - Progress tracking
   - Resource coordination

3. Executing Agents
   - Implementation details
   - Progress updates to Task-Rabbit
   - Resource requests through Task-Rabbit
   - Blockers reported to Task-Rabbit

4. Compliance & Security (ICE)
   - Security review
   - Compliance verification
   - Risk assessment

## Reporting Structure
- All progress updates → Task-Rabbit
- Task-Rabbit → Consolidated reports to Primo
- ICE → Security/compliance reports to Task-Rabbit

## Task Completion
1. Executing agent marks task complete
2. Task-Rabbit verifies completion
3. ICE performs compliance check
4. Task-Rabbit prepares final report
5. Primo reviews strategic objectives met
