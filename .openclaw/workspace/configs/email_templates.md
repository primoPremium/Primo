# Email Templates

## Default Template
```
Subject: Screenshot: {page_title}
Body:
Hi,

Please find attached the screenshot from {url}.

Captured on: {timestamp}

Best regards,
{sender_name}
```

## Report Template
```
Subject: Website Report: {page_title}
Body:
Hello,

Here's the latest screenshot from {url} for your review.

Capture Details:
- Date: {timestamp}
- URL: {url}
- Viewport: {viewport}

Notes: {notes}

Best regards,
{sender_name}
```

## Quick Share Template
```
Subject: Quick Screenshot Share
Body:
FYI - Screenshot attached.

{url}
Captured: {timestamp}
```
