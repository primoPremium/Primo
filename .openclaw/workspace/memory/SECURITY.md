# Security Rules

## Credential Handling
- NEVER share actual API keys, tokens, or credentials in chat messages
- Only reference environment variable names (e.g. WP_WOO_API_KEY) when needed
- Keep all sensitive values secure and hidden
- Use environment variables as intended - to protect sensitive data

## Documentation
- Only document the names of required environment variables
- Never include actual values in documentation or chat
- Use placeholders like $API_KEY or [YOUR_API_KEY] in examples
