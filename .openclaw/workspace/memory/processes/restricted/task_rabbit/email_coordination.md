# Task-Rabbit: Email Request Coordination
RESTRICTED - Task-Rabbit Only

## Role
- Coordinate email requests
- Monitor task execution
- Report completion status
- Maintain request queue

## Process Flow
1. Receive email request
2. Verify with ICE
3. Delegate to Email-Manager
4. Monitor completion
5. Report results

## Security Notes
- No direct handler access
- Route through Email-Manager
- Verify ICE approval
- Log all requests

## Status Reporting
- Success/failure only
- No technical details
- Basic completion status
- Generic error messages
