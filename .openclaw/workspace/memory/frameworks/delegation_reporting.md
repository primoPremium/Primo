# Delegation and Reporting Framework

## Core Principles
1. All responsibility lies with Primo (Marketing & Advertising Director)
2. No direct sub-agent reporting to chat
3. Clear documentation of all activities
4. Regular status updates
5. Cost-conscious resource management

## Delegation Process
1. Task Assessment
   - Clear objectives
   - Success metrics
   - Required resources
   - Expected timeline

2. Sub-agent Deployment
   - Specific instructions
   - Report back directly to Primo
   - Clear deliverables
   - Timeline constraints

3. Monitoring Process
   - Check progress every 5 minutes
   - Document findings
   - Compile results
   - Prepare summaries

## Reporting Structure
1. Progress Updates
   - Status: Not Started/In Progress/Completed
   - Key findings
   - Blockers/Issues
   - Next steps

2. Results Reporting
   - Executive summary
   - Detailed findings
   - Recommendations
   - Action items

3. Timeline Requirements
   - Initial confirmation within 2 minutes of task start
   - Progress update every 5 minutes
   - Final report within 2 minutes of completion
   - Follow-up actions within 24 hours

## Documentation Standards
1. All activities logged in memory/activities/YYYY-MM-DD/
2. Results documented in memory/results/
3. Critical findings flagged for immediate attention
4. Regular memory maintenance and organization

## Cost Management
1. Clear success criteria before deployment
2. Regular progress monitoring
3. Task termination if objectives unclear
4. Resource usage tracking
