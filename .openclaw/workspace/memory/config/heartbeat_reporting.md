# Heartbeat Reporting Configuration

## Update Frequency
- Limited to 4 updates per 24-hour period
- ONLY applies to heartbeat status updates
- Does NOT affect other notifications (task outputs, critical alerts, reports)

## Important Notes
- All scheduled tasks continue running as defined in HEARTBEAT.md
- Only the heartbeat status message frequency is limited
- All other system notifications, alerts, and reports continue at their normal frequency
- Critical alerts are always sent immediately regardless of timing
