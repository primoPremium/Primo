# Standard Reporting Format

## Requirements
1. Telegram Message Format:
   - Use emoji headers (🔄, 📅, ✅, etc.)
   - Apply Telegram markdown formatting (*italic*, bold)
   - Keep sections brief but informative
   - Use bullet points for better readability
   - Include timestamp and report ID

2. Here.now Component:
   - Create detailed markdown report
   - Include full metrics and details
   - Maintain professional formatting
   - Use consistent headers
   - Include timestamps and IDs matching Telegram post

3. Style Guide:
   - Colorful but professional
   - Concise yet thorough
   - Structured hierarchy
   - Clear section breaks
   - Easy to scan
   - Always include both summary (Telegram) and full (here.now) versions

## Template Structure
```
🔄 *Title*
📅 Date

*Key Sections*
✅ Status Points
✅ Important Metrics
✅ Critical Updates

*Active Systems*
• System 1: Status
• System 2: Status

*Recent Updates*
1. Update Category:
   - Detail Point
   - Detail Point

📊 *Status Summary*
Brief overview of current state

📑 Full Report: {here.now link}

Generated: {timestamp UTC}
Report ID: {unique-id}
```

## Usage Notes
- Always publish full report to here.now first
- Follow with formatted Telegram message
- Maintain consistent formatting across all reports
- Update template as needed based on feedback
