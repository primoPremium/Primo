# Example Case: Email Report Delegation

## Scenario
Request: "Can you email me (bweldy82@gmail.com) a copy of this report?"

## ❌ Incorrect Response (What Not To Do)
- Attempting to send email directly
- Using message tool with gmail channel
- Bypassing Task-Rabbit and Email-Manager

## ✅ Correct Flow
1. Primo (Marketing Director):
   - Recognizes email task
   - Delegates to Task-Rabbit
   - Provides clear task parameters

2. Task-Rabbit:
   - Receives delegation
   - Coordinates with Email-Manager
   - Monitors task progress
   - Reports completion status

3. Email-Manager:
   - Handles email composition
   - Manages delivery
   - Returns status to Task-Rabbit

## Key Learning Points
1. Never attempt direct email sending
2. Always follow delegation chain
3. Let Task-Rabbit coordinate between agents
4. Maintain proper reporting structure

## Template Delegation Message
```
Task-Rabbit: Please coordinate with Email-Manager agent to send the following email:

To: [recipient]
Subject: [subject]
Priority: [priority level]

Content: [details]

Please monitor delivery status and report back completion or any issues.
```
