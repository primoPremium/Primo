# Competitor Analysis Agent Configuration

## Purpose
Weekly automated analysis of competitor cannabis delivery websites in Orange County to benchmark features, offerings, and marketing strategies.

## Target Competitors
1. Primary Competitors (Direct Cannabis Delivery):
   - Eaze (eaze.com)
   - Grassdoor (grassdoor.com)
   - Amuse (amuse.com)
   - Hyperwolf (hyperwolf.com)

2. Secondary Competitors (Local Dispensaries with Delivery):
   - From-Locations dropdowns on our site

## Analysis Parameters
1. Website Performance
   - Page load times
   - Mobile responsiveness
   - Navigation structure
   - User experience elements

2. Marketing Elements
   - Hero section design
   - Call-to-action placement
   - Special offers presentation
   - Loyalty programs
   - New customer incentives

3. Product Presentation
   - Category organization
   - Product photography
   - Pricing display
   - Inventory indicators

4. Promotional Strategies
   - Daily/weekly deals
   - Holiday specials
   - Bundle offers
   - Minimum order incentives

5. Content Strategy
   - Blog/educational content
   - Product descriptions
   - SEO elements
   - Social proof integration

## Weekly Tasks
1. Screenshot competitor homepages
2. Document current promotions
3. Note new features/changes
4. Compare pricing on key products
5. Review mobile experience
6. Analyze special offers
7. Document delivery zones/times
8. Check payment options

## Report Structure
1. Executive Summary
2. Key Changes from Previous Week
3. Competitive Advantages/Threats
4. Recommended Actions
5. Screenshots Archive
6. Pricing Matrix Update

## Output Location
- Weekly reports: memory/competitor_analysis/YYYY-MM-DD_weekly_report.md
- Screenshots: memory/competitor_analysis/screenshots/YYYY-MM-DD/
- Pricing matrix: memory/competitor_analysis/pricing_matrix.md

## Security Notes
- Respect robots.txt
- No automated scraping

- Public-facing data only
