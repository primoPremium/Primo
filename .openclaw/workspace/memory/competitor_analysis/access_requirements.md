# Competitor Site Access Requirements & Procedures

## Access Methods

### Direct Browser Access (Consumer Approach)
- Use regular consumer browser
- Access through mobile device when possible
- Use residential IP address
- Complete age verification normally
- Enter valid delivery address
- Allow 2-3 seconds between actions

### Alternative Research Methods
1. Mobile Apps (when available)
2. Public marketing materials
3. Social media presence
4. Customer reviews/feedback
5. Newsletter subscriptions

## Competitor-Specific Requirements

### Eaze.com
Access Requirements:
- Cloudflare protection in place
- Requires residential IP
- Age verification required
- Location verification required
Best Method: Mobile app or residential access

### Grassdoor
Access Requirements:
- Domain verification needed
- Location-based access
- Age verification required
Best Method: Mobile app or residential access

### Amuse
Access Requirements:
- Standard age verification
- Location verification
- Less strict bot protection
Best Method: Direct website access with location

### Hyperwolf
Access Requirements:
- Age verification
- Location-based menu
- Account creation for full access
Best Method: Account creation from residential IP

## Standard Access Procedure

1. Initial Visit
   - Use residential IP or mobile network
   - Allow page to fully load (15-20 seconds)
   - Accept all necessary cookies
   
2. Age Verification
   - Complete when prompted
   - Use standard "Yes/No" options
   - Wait 2-3 seconds between actions

3. Location Verification
   - Use valid Orange County address
   - Within delivery radius
   - Save location if prompted

4. Navigation
   - Natural browsing speed
   - 2-3 seconds between clicks
   - Use normal scroll speed
   - Don't open multiple tabs simultaneously

5. Data Collection
   - Screenshot key pages
   - Note prices and promotions
   - Document delivery fees/minimums
   - Save terms and conditions
